## Endpoint

/v1/audio/transcriptions
Authorization

Body

## Authorization
| Name | Type | Required | Description |
|---|---|---|---|
| Authorization | string | yes | AuthorizationBearer <token> 使用 Bearer Token 驗證。 格式: `Authorization: Bearer sk-xxxxxx`.. In: `header`.. |
## Request Body

| Name | Type | Required | Description |
|---|---|---|---|
| file | file | yes | 音訊檔案 Format`binary`.. |
| model | string | yes |  |
| language | string | no | ISO-639-1 语言代码 |
| prompt | string | no |  |
| response_format | string | no | Default`"json"`.. Value in`"json" \| "text" \| "srt" \| "verbose_json" \| "vtt"`.. |
| temperature | number | no |  |
| timestamp_granularities | array<string> | no |  |
