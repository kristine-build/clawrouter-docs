# 文本转語音

将文本转换为音訊



## Endpoint

/v1/audio/speech
Authorization

Body

## Authorization
| Name | Type | Required | Description |
|---|---|---|---|
| Authorization | string | yes | AuthorizationBearer <token> 使用 Bearer Token 驗證。 格式: `Authorization: Bearer sk-xxxxxx`.. In: `header`.. |
## Request Body
application/json

| Name | Type | Required | Description |
|---|---|---|---|
| model | string | yes |  |
| input | string | yes | 要转换的文本 Length`length <= 4096`.. |
| voice | string | yes | Value in`"alloy" \| "echo" \| "fable" \| "onyx" \| "nova" \| "shimmer"`.. |
| response_format | string | no | Default`"mp3"`.. Value in`"mp3" \| "opus" \| "aac" \| "flac" \| "wav" \| "pcm"`.. |
| speed | number | no | Default`1`.. Range`0.25 <= value <= 4`.. |
