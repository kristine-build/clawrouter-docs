# 原生OpenAI格式

取得當前可用的模型清單。

根据請求头自动辨識回傳格式：

- 包含 `x-api-key`.. 和 `anthropic-version`.. 头时回傳 Anthropic 格式
- 包含 `x-goog-api-key`.. 头或 `key`.. 查詢參數时回傳 Gemini 格式
- 其他情况回傳 OpenAI 格式



## Endpoint

/v1/models
Authorization

Header

Query

## Authorization
| Name | Type | Required | Description |
|---|---|---|---|
| Authorization | string | yes | AuthorizationBearer <token> 使用 Bearer Token 驗證。 格式: `Authorization: Bearer sk-xxxxxx`.. In: `header`.. |
## [Query Parameters](#query-parameters)

key?string

Google API Key (用于 Gemini 格式)

## [Header Parameters](#header-parameters)

x-api-key?string

Anthropic API Key (用于 Claude 格式)

anthropic-version?string

Anthropic API 版本

x-goog-api-key?string

Google API Key (用于 Gemini 格式)
