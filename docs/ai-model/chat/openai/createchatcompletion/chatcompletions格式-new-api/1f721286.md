# ChatCompletions格式

根据對話历史建立模型回應。支援流式和非流式回應。

兼容 OpenAI Chat Completions API。



## Endpoint

/v1/chat/completions
Authorization

Body

## Authorization
| Name | Type | Required | Description |
|---|---|---|---|
| Authorization | string | yes | AuthorizationBearer <token> 使用 Bearer Token 认证。 格式: `Authorization: Bearer sk-xxxxxx`.. In: `header`.. |
## Request Body
application/json

| Name | Type | Required | Description |
|---|---|---|---|
| model | string | yes | 模型 ID |
| messages | array<object> | yes | 對話消息清單 |
| temperature | number | no | 采样温度 Default`1`.. Range`0 <= value <= 2`.. |
| top_p | number | no | 核采样參數 Default`1`.. Range`0 <= value <= 1`.. |
| n | integer | no | 生成数量 Default`1`.. Range`1 <= value`.. |
| stream | boolean | no | 是否流式回應 Default`false`.. |
| stream_options | object | no |  |
| stop | string\|array<string> | no | 停止序列 |
| max_tokens | integer | no | 最大生成 Token 数 |
| max_completion_tokens | integer | no | 最大補全 Token 数 |
| presence_penalty | number | no | Default`0`.. Range`-2 <= value <= 2`.. |
| frequency_penalty | number | no | Default`0`.. Range`-2 <= value <= 2`.. |
| logit_bias | object | no |  |
| user | string | no |  |
| tools | array<object> | no |  |
| tool_choice | string\|object | no |  |
| response_format | object | no |  |
| seed | integer | no |  |
| reasoning_effort | string | no | 推理强度 (用于支援推理的模型) Value in`"low" \| "medium" \| "high"`.. |
| modalities | array<string> | no |  |
| audio | object | no |  |
