# 原生Claude格式

Anthropic Claude Messages API 格式的请求。
需要在请求头中包含 `anthropic-version`。



## Endpoint

/v1/messages
Authorization

Header

Body

## Authorization
| Name | Type | Required | Description |
|---|---|---|---|
| Authorization | string | yes | AuthorizationBearer <token> 使用 Bearer Token 认证。 格式: `Authorization: Bearer sk-xxxxxx` In: `header` |
## [Header Parameters](#header-parameters)

anthropic-version\*string

Anthropic API 版本

x-api-key?string

Anthropic API Key (可选，也可使用 Bearer Token)
